# Voice Interview Bot

A minimalist voice interview assistant with a Siri-style animated orb UI, powered by OpenAI's Realtime API.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```

2. Create `.env` file:
   ```bash
   cp .env.example .env
   ```

3. Add your OpenAI API key to `.env`

4. Start the server:
   ```bash
   npm start
   ```

5. Open http://localhost:3000

## Customize

Edit `server.js` to update the `personalInfo` object with your own details.
